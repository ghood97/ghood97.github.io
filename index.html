Facture

I built a C# Windows Presentation Foundation app for my first Full-Stack application on the .NET Framework. Facture is an invoice creation app that lets you create invoices from items you already have in your inventory. In addition to C#, I used XAML for the front-end and SQLite for the backend. I enjoyed using C# because of its strict typing. This makes it easier to track down the source of errors.
Technologies used: C#, XAML, SQLite, Windows Presentation Foundation

TeamNotiphy

My first deployed React application is called TeamNotiphy, an application for sports teams. The inspiration for this app came from the Needham High School Wrestling Team. As a coach of this team, I needed to create a central hub for communication between athletes, coaches and parents. Members of the teams organization can sign-in to see updates about the team and view upcoming events. Members can also comment on notifications and view the teams competition results.
Technologies used: React, Ruby on Rails, React-Bootstrap
